file path : đường dẫn file 
        áp dụng cho đường dẫn của hình ảnh thẻ img 
        áp dụng đường dẫn của file css 
        áp dụng đường dẫn của file javascript 
        áp dụng đường dẫn của file html khác ... 
        có 4 loại chính ( học 3 loại )
        1 - đường dẫn cần nhúng chung thư mục với file html 
        2 - đường dẫn cần nhúng nằm trong folder chung thư mục với file html 
        3 - đường dẫn cần nhúng nắm ngoài thư mục chứa file html
