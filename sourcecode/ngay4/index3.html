thuộc tính css position ( vị trí )

        mặc định các thẻ html có position là static ( nghĩa là tĩnh )
        khi ta thay đổi giá trị cho thuộc tính position thì tính chất của 
        thẻ đó sẽ bị thay đổi 
        1 - position: static; ( mặc định là tĩnh )
        2 - position: relative; ( là tương đối )
        3 - position: absolute; ( là tuyệt đối )
        4 - position: fixed; ( cố định )
        5 - position: sticky; ( kết hợp fixed và relative )

        dùng 1 số thuộc tính tọa độ trong css như top, left, right, bottom
.demoA static
.demoB absolute
.demoC absolute
.demoD absolute
.demoE absolute
