thuộc tính css  ( khi tràn nội dung thì sẽ xử lý ra sao ?)
        : visible; ( hiển thị phần tràn)
        : hidden; ( ẩn phần tràn )
        : scroll; ( hiển thanh cuộn chuột trong cả 2 trường hợp nội dung tràn hay không tràn)

        -x ( xử lý tràn theo chiều ngang )
        -y ( xử lý tràn theo chiều dọc )
