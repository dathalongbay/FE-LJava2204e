file path : đường dẫn file 
        đường dẫn file có 2 trường hợp chính 
        1 - file path tuyệt đối : ví dụ đường dẫn của tên miền 
        2 - file path tương đối : lại chia thành nhiều trường hợp nhỏ hơn 
        file path áp dụng 
        đường dẫn của hình ảnh 
        đường dẫn của file css 
        đường dẫn của file html khác 
        đường dẫn của file javascript ...
