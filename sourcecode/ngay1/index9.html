thẻ input trong form 
        - thẻ label là thẻ nhãn văn bản trước thẻ input
        - input type="text" chứa văn bản có thuộc tính name là tên của ô input . giá trị của name nên viết liền không dấu 
        value="" là giá trị của input đó 
        - thuộc tính placeholder="" trong thẻ input gới ý cho người dùng biết phải nhập gì vào trong input

        thẻ input có type="email"
        khi submit thì trình duyệt sẽ kiểm tra trong thẻ đó có phải định dạng email hay không 


        thẻ input type="radio" có thuộc tính chọn trước 1 trong các option radio là thuộc tính checked

        thẻ select chứa các thẻ option 
        thẻ option có thuộc tính là value
        nếu muốn chọn trước 1 option làm mặc định thì như thế nào 
        mặc định thì trình duyệt sẽ chọn option đầu tiên trong select làm mặc định . dùng thuộc tính selected trong option

        thẻ textarea giống như thẻ input text ngoại trừ thẻ textarea chứa được nhiều dòng 
        cols="30" rows="10"
        cols là số cột của textarea
        rows là số hàng hay số dòng của textarea


        input type là checkbox
        nếu muốn mặc định checkbox đã được check vào thì thêm thuộc tính checked vào

        input type="submit" khi click vào sẽ submit form đến đích cần gửi
