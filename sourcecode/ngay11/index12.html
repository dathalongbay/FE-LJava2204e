JSON là kiểu định dạng dữ liệu để trao đổi giữa server và client 
        hoặc giữa server này với server kia 
        ngoài ra JSON dùng lưu trữ dư liệu 

        kiểu dữ liệu json tương tự như js object nhưng khác nhau 
        {"name": "Peter", "age": 22, "country": "United States"}
        có key:value bên trong {} và nó là 1 chuỗi 
        dạng mảng json trong []
        ví dụ :
        [
        {"name": "Peter", "age": 22, "country": "United States"},
        {"name": "Peter", "age": 22, "country": "United States"},
        {"name": "Peter", "age": 22, "country": "United States"}
        ]
