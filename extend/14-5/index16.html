thuộc tính style trong thẻ html được dùng để trang trí css cho thẻ đó 
        cú pháp là style="mã nguồn css"
        ví dụ :
        style="color:red;background-color: green;"
        trong mã css có thuộc tính css và giá trị của nó
        cú pháp là ten_thuoc_tinh:gia trị;ten_thuoc_tinh2: giá trị
        color là màu chữ 
        background-color là màu nền
        font-size là kích cỡ văn bản
Thẻ div với css style
