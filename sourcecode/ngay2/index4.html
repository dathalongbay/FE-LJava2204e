3 cách nhúng css vào trong 1 file html 
        cách 1 : inline style ( sử dụng thuộc tính style trong thẻ mở của thẻ html)
        cách 2 : internal style ( sử dụng thẻ style bên trong thẻ head của file html)
        cách 3 : external style ( sử dụng file .css từ bên ngoài và nhúng link vào file html ) dùng thẻ link có thuộc tính rel="stylesheet" và href="đường dẫn của file css"
