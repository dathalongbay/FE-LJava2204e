nếu dùng display:inline thì việc set width và height mất tác dụng 
        hay nói cách khác ko thể set width và height cho 1 thẻ có display là inline 
        => display: inline-block; 
        áp dụng cho trường hợp khi muốn đặt width và height
         cho 1 thẻ mà vẫn muốn giữ đặc tính trên cùng 1 hàng của inline
